CRACKJACK - Music Folder
=======================

Add your soundtrack file here!

Supported formats:
- MP3 (soundtrack.mp3) - Recommended
- OGG (soundtrack.ogg) - Alternative
- WAV (soundtrack.wav) - Uncompressed

The game will automatically load (in order):
1. soundtrack.mp3
2. soundtrack.ogg
3. soundtrack.wav

Or use the in-game Options menu to load any audio file!

Recommended: Add some casino/lounge music or 
something hilariously dramatic for maximum effect.
